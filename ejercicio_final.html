Flexbox
Grid
Imagen

Modelo unidimensional para alinear y distribuir elementos en un solo eje (fila o columna).

Modelo bidimensional que permite controlar filas y columnas simultáneamente para diseños complejos.

Características:
  • Alineación y distribución en un eje
  • Ideal para menús y barras
  • Controles: justify-content, align-items, flex-direction
Características:
  • Control de filas y columnas
  • Definición de áreas (grid-template-areas)
  • Ideal para layouts complejos
